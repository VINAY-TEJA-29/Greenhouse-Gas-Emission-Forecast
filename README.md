# 🌍 Greenhouse Gas Emission Forecasting

This project is part of the AICTE – Green AI Internship (June 2025 Cycle), aiming to predict future greenhouse gas emissions using historical datasets and machine learning.

## 🎯 Objective
To forecast future GHG emissions (like CO₂, CH₄) for industries using ARIMA time series models.

## 🗂️ Project Files
- `week_1/main.py`: Code for reading and checking dataset
- `week_1/ghg_data.csv`: Dataset from Edunet Foundation
- `README.md`: Project summary

## 🛠️ Tools & Libraries
- Python, Pandas, Matplotlib
- ARIMA (via `pmdarima`)
- Jupyter / VS Code
- GitHub (for version control)

## 🔮 Week Plan
- **Week 1**: Dataset understanding and project setup
- **Week 2**: ARIMA modeling and forecasting
- **Week 3**: Visualization and optimization
- **Week 4**: Report, PPT, and Final Submission
